# book-insights-ai
A Python application that scrapes book data from the web and generates AI-powered insights.

## What it does
- Scrapes 20 books from books.toscrape.com
- Generates AI summaries for each book
- Predicts book genres
- Gives book recommendations

## Files
- `Untitled0.ipynb` - Main Python code (Google Colab notebook)
- `books_data.json` - Scraped book data with AI insights

## How to run
1. Open `Untitled0.ipynb` in Google Colab
2. Run all cells in order
3. Books data will be saved to `books_data.json`

## Sample Output
- Book: A Light in the Attic
- Genre: Thriller
- Summary: A captivating book with a Three star rating
- Recommendation: If you liked this, try more Thriller books!

## Tech Used
- Python
- BeautifulSoup (web scraping)
- Google Gemini AI API
